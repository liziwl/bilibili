## 数据分析抓包
### 因变量（Y）
- [x] b站视频播放量
### 自变量（X）
- [x] 视频名称（与视频号码一一对应）
- [x] 上传时间
- [x] 视频类型
- [x] 视频标签
- [x] 视频获得的硬币量
- [x] 视频获得的收藏次数
- [x] 视频获得的分享次数
- [x] 视频时长
- [ ] up主（视频上传者）投稿数量
- [ ] up主（视频上传者）粉丝数。
